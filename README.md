# Claude-Sim

Claude-Sim is a light theme setup for Obsidian and Codex, tuned toward the low-saturation near-white look of Claude Desktop.

## Included

- `obsidian/theme.css`: Obsidian theme CSS.
- `obsidian/manifest.json`: Obsidian theme manifest.
- `codex/codex-theme-v1.txt`: Codex import string.
- `codex/config-snippet.toml`: Codex config snippet for `~/.codex/config.toml`.
- `fonts.md`: Font dependencies and fallback order.

## Obsidian

Install by copying the `obsidian` folder contents into:

```text
<vault>/.obsidian/themes/Claude-Sim/
```

Then select `Claude-Sim` in Obsidian's Appearance settings.

## Codex

Use the import string in `codex/codex-theme-v1.txt`, or copy the TOML snippet in `codex/config-snippet.toml` into `~/.codex/config.toml`.
